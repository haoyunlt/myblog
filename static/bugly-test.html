💡 故障排查提示

如果仪表板没有数据，可能的原因：
  1. 你在桌面浏览器访问（Bugly 默认只在移动设备启用）
  2. 没有发生任何错误（系统运行正常）
  3. bugly-report.js 脚本未正确加载
  4. localStorage 被清除或禁用

解决方案：

  1. 使用移动设备访问或在开发者工具中启用移动模拟
  2. 点击上方的测试按钮生成测试错误
  3. 检查浏览器控制台确认脚本已加载
  4. 确保浏览器允许 localStorage
